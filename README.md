# VTuber Task Manager

Flaskで作成したシンプルなタスク管理Webアプリです。

## 🔧 使用技術
- Python
- Flask
- HTML / CSS
- JSON（データ永続化）
- Git / GitHub

## ✨ 主な機能
- タスクの追加
- 編集
- 削除
- 完了チェック
- JSONによるデータ保存
