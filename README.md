# VTuber Task Manager

VTuber向けタスク管理アプリです。  
Flaskを使って作成されたシンプルなWebアプリで、タスクの追加・編集・完了・削除ができます。

## 📌デモ
http://127.0.0.1:5000/
 
## 🎯 主な機能

- タスクの追加／削除
- タスクの編集
- 完了チェック（チェック済み表示）
- 永続化(JSON保存)

## 🛠 使用技術

- Python
- Flask
- Jinja2
- HTML / CSS
